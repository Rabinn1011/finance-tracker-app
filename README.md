# Finance Tracker App

A Flutter-based personal finance tracking application - your digital budget notebook.

## What is this?
Instead of tracking your income, expenses, and money in a physical notebook with pen and paper,
use this app! Manually log your transactions and see where your money is going.

## Features (Planned)
-  Track income and expenses
- Organize money across different accounts (eSewa, Khalti, Bank, Cash)
-  Visual analytics and spending insights
-  Budget management
- See where your money goes with charts
-  Secure authentication with Supabase

## Tech Stack
- Flutter
- Supabase (Backend & Auth)
- Provider (State Management)
- GoRouter (Navigation)

1. Clone the repository
2. Run `flutter pub get`
